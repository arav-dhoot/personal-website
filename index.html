Projects
I have worked on multiple projects over the past few years. So as to not bore you with the details, I will briefly describe a few projects that I have found most fulfilling.

  1. COMForce: It is the first device to detect the quality and maturity of compost simultaneously. I used a computational approach to provide reliable information to farmers quickly. I had to learn how to use RPi for this project. While then tiring, it was a rewarding process building the entire dataset to scratch.
  2. Remote PD Detection: I built a program to detect and classify the severity of Parkinson's Disease using in-the-wild video (first program to achieve this) while achieve state of the art performance. I had a lot of fun working on this project and presenting it and collecting the award for it at ISEF (subtle brag, I know).
  3. Transformer Optimization: I trained GPT-2 on GLUE tasks and monitored the gradients of each of the layers while training. To improve computational efficiency while maintaining performance, I trained an optimized model (freezing layers with lower variance and, thus, learning). I achieved some interesting and promising results. Shoot me an email to discuss more about this project.
